Completed the **Google Data Analytics Professional Certificate**, an 8-course program designed to prepare learners for entry-level Data Analyst roles. The program focused on how to prepare, process, analyze, and share data insights effectively.

## Skills Gained

Developed proficiency in industry-standard tools including **spreadsheets**, **SQL**, **Tableau**, and **R programming** for data analysis and visualization.

## Course Curriculum

- Foundations: Data, Data, Everywhere
- Ask Questions to Make Data-Driven Decisions
- Prepare Data for Exploration
- Process Data from Dirty to Clean
- Analyze Data to Answer Questions
- Share Data Through the Art of Visualization
- Data Analysis with R Programming
- Google Data Analytics Capstone: Complete a Case Study

## Key Takeaways

This certification provided hands-on experience with real-world data analysis scenarios, from data cleaning and processing to creating meaningful visualizations and presenting insights to stakeholders.
