Completed the **Google IT Automation with Python Professional Certificate**, a 6-course program designed to prepare learners for entry-level IT roles with a focus on Python programming and infrastructure automation.

## Skills Gained

Developed proficiency in **Python programming**, **Git and GitHub**, **configuration management**, **cloud infrastructure**, **debugging techniques**, and creating **automation scripts** for real-world IT tasks.

## Course Curriculum

- Crash Course on Python — foundational syntax, data types, loops, and functions
- Using Python to Interact with the Operating System — file handling, subprocesses, and automation scripts
- Introduction to Git and GitHub — version control fundamentals and collaborative workflows
- Troubleshooting and Debugging Techniques — systematic problem-solving and error handling
- Configuration Management and the Cloud — using Puppet and cloud tools for scalable infrastructure
- Automating Real-World Tasks with Python — building end-to-end automation solutions for practical IT tasks

## Key Takeaways

This certification provided hands-on experience building automation scripts to streamline IT operations, from simple file management to complex infrastructure configuration. Each module included practical labs and projects reinforcing version control best practices, systematic debugging approaches, and scalable automation patterns used in production environments.
